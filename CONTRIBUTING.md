# Contributing to Czkawka macOS Guide

Thanks for your interest in improving this guide! This repository is specifically for macOS installation and usage documentation.

## Important Notes

- **This is NOT the official Czkawka project.** For issues with the Czkawka application itself, please visit the [official repository](https://github.com/qarmin/czkawka).
- This repository contains only documentation, not application code or binaries.

## How to Contribute

### Reporting Issues

If you encounter problems with the installation instructions or find errors in the guide:

1. Check if the issue already exists in the [Issues](../../issues) section
2. If not, create a new issue with:
   - Clear description of the problem
   - Your macOS version and chip type (Intel/Apple Silicon)
   - Steps to reproduce (if applicable)
   - Screenshots if helpful

### Suggesting Improvements

We welcome suggestions for:
- Clearer installation instructions
- Additional troubleshooting tips
- Better formatting or organization
- Corrections to technical details

### Submitting Changes

1. Fork this repository
2. Create a new branch (`git checkout -b improve-installation-steps`)
3. Make your changes
4. Test the instructions if possible
5. Commit your changes (`git commit -am 'Add clearer permission instructions'`)
6. Push to the branch (`git push origin improve-installation-steps`)
7. Create a Pull Request

### Guidelines

- Keep instructions clear and concise
- Test commands before submitting (when possible)
- Maintain the existing formatting style
- Always link to official Czkawka releases, never redistribute binaries
- Ensure proper attribution to the original Czkawka project

## Questions?

If you have questions about using Czkawka itself, please refer to the [official Czkawka repository](https://github.com/qarmin/czkawka).

For questions about this guide, feel free to open an issue.

Thank you for helping make this guide better!
